# 🐍 Snake Game (Python + Pygame)

A classic Snake game built with **Python** and **Pygame**. 
This project demonstrates basic game development concepts like movement, collision detection, and event handling.

---

## 🎮 Features

- Smooth snake movement
- Random food generation
- Snake grows after eating food
- Game resets on:
  - Wall collision
  - Self collision
- Press **Q** to quit the game

---

## 🛠️ Technologies Used

- Python 3.x
- Pygame

---

## ▶️ How to Run

```bash
python snakegame.py
```

---

## 🎯 Controls

| Key | Action |
|------|--------|
| W | Move Up |
| S | Move Down |
| A | Move Left |
| D | Move Right |
| Q | Quit Game |

---

## 🧠 Gameplay

- The snake moves in fixed tile increments.
- Food appears randomly inside the window. Guide the snake to eat the red food tiles to grow longer.
- The snake grows by one segment each time it eats food.
- Avoid hitting the walls or the snake's own body — doing so resets the game.
- Food respawns in a new random location after being eaten
